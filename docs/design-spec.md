# 设计规范

## 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景色 | #f0f5fa | 淡蓝灰，主背景 |
| 面板背景 | #e3edf5 | 标题栏背景 |
| 卡片背景 | #ffffff | 白色卡片 |
| 主色调 | #4a7aa9 | 深蓝，文字/图标 |
| 主色浅 | #7aadd6 | 浅蓝，交互高亮 |
| 主色底 | #eaf3fa | 极浅蓝，标签底色 |
| 边框色 | #d0dde8 | 浅灰蓝边框 |
| 文字色 | #333333 | 正文 |
| 副文字 | #8aa0b5 | 辅助信息 |

## 布局
- 窗口默认尺寸：420×640 px
- 最小尺寸：320×400 px
- 无边框窗口，自定义标题栏
- 卡片间距：8px
- 卡片内边距：12px
- 卡片圆角：12px

## 字体
- 系统：Microsoft YaHei / PingFang SC
- 标题：14px bold
- 正文：13px regular
- 辅助：12px regular

## 交互
- 卡片悬停：轻微上浮阴影 + 边框加深
- 点击卡片：复制并粘贴，窗口关闭
- 置顶卡片：浅蓝色边框 + 📌 图标
- 删除按钮：红色（仅悬停显示）
