# 开发需求文档

## 项目名称
历史粘贴板 (Clipboard History)

## 目标用户
Windows 普通用户，无需编程知识即可使用。

## 核心功能

### 1. 剪切板监控
- 后台持续监控系统剪切板
- 支持文本和图片两种格式
- 自动记录每次复制的内容
- 去重：相同内容不重复记录

### 2. 历史记录管理
- 以时间降序排列展示所有历史
- 每条记录以卡片形式展示，含预览和时间戳
- 点击记录自动粘贴到当前活动窗口
- 支持置顶重要记录
- 支持删除不需要的记录
- 全文搜索（匹配文本内容）

### 3. 存储管理
- 可配置存储期限：1天 / 3天 / 5天
- 到期自动清理过期记录
- 图片以文件形式存储，数据库仅存路径

### 4. 界面交互
- Win+V 全局快捷键唤起
- 系统托盘图标常驻
- 窗口失去焦点自动关闭
- 淡蓝色简洁 UI 设计

### 5. 系统集成
- 开机自启动
- 最小化到系统托盘
